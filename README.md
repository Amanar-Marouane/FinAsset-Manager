## Installation

1. Install dependencies:
   ```bash
   npm i
   ```

2. Create a `.env` or `.env.production` file in the project root with the following content:
   ```
    NEXT_PUBLIC_API_PROVIDER=http://localhost:8000/api
    NEXT_PUBLIC_APP_NAME="NextEdge"
    NEXT_PUBLIC_IMG_PROXY=http://localhost:8000/api/proxy/
   ```
